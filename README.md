# About the Game

Control the snake with simple taps!

Retro-Snake is a clean and modern take on the timeless classic, built entirely in Java using Android Studio.
Designed to be lightweight and responsive, the game runs smoothly on nearly all Android devices.

Move the snake using on-screen direction buttons as it glides across a grid. Every time you grab a piece of food, the snake grows longer and the challenge increases. One wrong move and the game ends!
